# 雀语图书馆手机端网页

雀语图书馆是一个现代化的图书馆管理系统，专为移动设备优化设计。该系统提供了用户友好的界面，支持图书浏览、借阅、归还等功能。

## 项目概述

这是一个纯前端项目，使用HTML、CSS和JavaScript构建，无需后端支持即可运行。用户数据通过浏览器本地存储(LocalStorage)进行管理。

## 技术栈

- HTML5
- Tailwind CSS
- JavaScript (ES6+)
- Font Awesome 图标库
- 本地存储 (LocalStorage)

## 功能模块

### 1. 用户认证模块

#### 登录页面 (login.html)
- 用户身份验证
- 记住我功能
- 密码可见性切换
- 测试账号支持

#### 注册页面 (register.html)
- 新用户注册
- 用户类型选择（学生/教职工）
- 表单验证（密码强度、一致性检查）

### 2. 主页面模块 (待开发)

主页面是系统的核心功能区域，将包含以下功能：

- 借阅记录查看

- 图书借阅/归还操作

### 3. 数据存储模块

使用浏览器本地存储管理所有数据：
- 用户信息存储

## 主页面实现方案

### 页面布局设计

主页面将采用响应式设计，包含以下几个主要区域：
1. 顶部导航栏：包含Logo、用户信息和通知
2. 搜索区域：快速查找图书
3. 分类导航：按类别浏览图书
4. 图书展示区：网格或列表形式展示图书
5. 底部导航：首页、分类、借阅记录、个人中心

### 核心功能实现

#### 1. 图书浏览功能
- 使用卡片式布局展示图书信息
- 支持按分类筛选
- 支持搜索功能
- 图书详情弹窗展示

#### 2. 借阅功能
- 点击借阅按钮发起借阅请求
- 本地存储借阅记录
- 显示可借阅状态和数量

#### 3. 归还功能
- 在借阅记录中查看已借图书
- 点击归还按钮完成归还操作
- 更新库存状态

#### 4. 收藏功能
- 将喜欢的图书添加到收藏夹
- 支持取消收藏
- 收藏夹独立页面展示

1. 本系统使用浏览器本地存储，清除浏览器数据会丢失所有信息
2. 不同浏览器之间数据不共享
3. 适合演示和学习用途，生产环境建议增加后端支持
